# 学习路线

二次根式是一种代数式 (opens new window)，如果你看到2\sqrt 22​、a\sqrt aa​、23\sqrt \frac {2}{3}32​​或者ax+b\sqrt {ax+b}ax+b​、ax2+bx+c\sqrt {ax^2+bx+c}ax2+bx+c​、2x−62x+6\sqrt \frac {2x-6}{2x+6}2x+62x−6​​等等——你基本上可以确定，这些代数式就是二次根式。

二次根式如同可以化简为整数 (opens new window)的分数一样，即使可以化简为整数，在没有化简 (opens new window)之前依然是二次根式，如4\sqrt 44​。

二次根式如同整式 (opens new window)中的单项式 (opens new window)一样，多个二次根式的组合式，不叫二次根式，如a+1⋅a−1\sqrt {a+1} \cdot\sqrt {a-1}a+1​⋅a−1​。

# 学习路线

二次根式属于代数式中的基础知识，产生于实数平方运算的逆运算-求平方根，需要具备实数相关知识，因此在学习二次根式之前，我们强烈建议你首先掌握上述知识，如果你不太熟练上述知识，请从以下模块开始学习之旅：

  • 代数 (opens new window)
  • 有理数与实数 (opens new window)
  • 整式与分式 (opens new window)

熟悉二次根式的基本性质之后，你将可以具备学习更多高级主题的能力，比如说：

  • 深入理解二次根式，如二次根式考试指南 (opens new window)中的知识
  • 指数函数 (opens new window)
